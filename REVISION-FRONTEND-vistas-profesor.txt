QUÉ CAMBIÓ RESPECTO DE LAS MAQUETAS DE HOME (y por qué)
--------------------------------------------------------------------------------
  - Las maquetas de Home eran HTML estático autónomo (traían su propio <head>,
    header y sidebar, y datos hardcodeados). Las vistas nuevas se integraron al
    _Layout.cshtml (header/sidebar/CSS/JS ya vienen del layout) y son DINÁMICAS
    (reciben datos por @model con DTOs). Por eso NO llevan <html>/<head>.
  - Se reutilizaron EXACTAMENTE las mismas clases CSS del proyecto para mantener
    el diseño. No se agregó CSS nuevo.
